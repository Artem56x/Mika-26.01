Mika – персональный Python-ассистент

Mika – это простой персональный ассистент на Python, который помогает с повседневными задачами, предоставляет информацию, развлечения и позволяет управлять напоминаниями.

📝 Возможности
Открытие сайтов и сервисов:
Google, YouTube, новости, музыка, кино и др.
Поиск в интернете:
Поиск в Google: найди <текст>
Поиск на YouTube: ютуб <текст>

Дата и время:
Узнать текущую дату: какое сегодня число
Узнать день недели: день недели
Узнать текущее время: сколько сейчас времени

Мини-игры и развлечения:
Монетка: монетка

Команды справки:
команды – выводит список всех доступных команд

⚡ Пример использования
Вы: открой ютуб
Mika: Открываю сайт...

Вы: найти Python уроки
Mika: Ищу в Google: Python уроки

Вы: монетка
Mika: Орёл

💻 Установка и запуск
Клонируйте репозиторий или скачайте код.
Убедитесь, что установлен Python 3.x.

Запустите программу:
python main.py
Следуйте инструкциям в консоли.
